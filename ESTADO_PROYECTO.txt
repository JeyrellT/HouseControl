FECHA: Abril 2026

================================================================================
1. ¿QUÉ ES?
================================================================================
Dashboard inteligente para controlar dispositivos del hogar (luces, puertas, 
cámaras, parlantes) con UI adaptativa basada en carga cognitiva del usuario.
Tecnología innovadora: detección de señales ErrP (neuro-adaptatividad).

================================================================================
2. FUNCIONALIDAD ACTUAL - LO QUE FUNCIONA ✅
================================================================================

CARACTERÍSTICAS COMPLETAS:
────────────────────────
• Dashboard con KPIs (dispositivos online, salud de plataformas)
• Gestión de dispositivos por tipo (Luces, Puertas, Cámaras, Parlantes)
• Organización de dispositivos por habitaciones
• Timeline de actividad en tiempo real
• Panel de escenas rápidas (one-click)
• Panel de acciones recomendadas con prioridades

DISPOSITIVOS SOPORTADOS:
• Luces: encender/apagar + controlar brillo
• Puertas: estado de candado + batería + indicador seguridad
• Cámaras: grabación + detección movimiento + modo privacidad
• Parlantes: estado reproducción + volumen

FUNCIONES COGNITIVAS (Neuromorphic):
• Detección de carga cognitiva (Alto/Normal/Bajo)
• Detección de señales de error (ErrP flag)
• UI adaptativa: simplificación automática en carga alta
• Modos de densidad: Comfort vs Focus
• Temas: Light/Dark mode con persistencia

MOTOR DE RECOMENDACIONES:
• Sistema basado en reglas
• Scores de confianza y prioridad
• Explicaciones a usuario
• Opción de descartar recomendaciones

INFRAESTRUCTURA:
✅ State Management centralizado (Nexus Context + Reducer)
✅ Architecture Domain-Driven
✅ Mock data completo para desarrollo
✅ Tests con Vitest + React Testing Library
✅ LocalStorage persistence
✅ Diseño responsive + accesibilidad (ARIA)
✅ Tokens de diseño CSS
